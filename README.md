# Heart Disease Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze heart disease survival outcomes using clinical data.

## Objectives
- Analyze survival rate across different age groups
- Identify key health indicators affecting mortality
- Create interactive and user-friendly dashboards

## Tools Used
- Power BI
- DAX
- MS Excel

## Key Insights
- Survival Rate by Age Group
- Gender-based survival analysis
- Impact of serum creatinine and ejection fraction
- KPI cards for quick decision-making
